HTML section Element

The section element defines a section in a document.
According to W3C's HTML documentation: "A section is a thematic grouping of content, typically with a heading."
A web page could normally be split into sections for introduction, content, and contact information.

Articles Tag

The article element specifies independent, self-contained content. An article should make sense on its own, and it should be possible to distribute it independently from the rest of the web site. Examples of where an article element can be used:

  • Forum Post
  • Blog post
  • NewsPaper article
